# Attendify - Club Event Attendance tracker

Attendify is designed to simplify how college events are managed and experienced. It solves common issues like unorganized attendance tracking, delayed certificate distribution, and lack of centralized event data.

The platform provides a structured and efficient way for both clubs and students to interact with events.

## Features

- #### Event Management

  Clubs can create, manage, and monitor events in a centralized system without relying on manual processes.

- #### Attendance Tracking

  Attendance is recorded digitally, ensuring accuracy and easy verification without paperwork.

- #### Certificate Generation

  Participants who attend events can receive certificates without delays or manual follow-ups.

- #### Student Interface

  Students can:
  - Register for events
  - Mark attendance
  - Access their certificates
